# cosmic-shapes
Simple canvas cordova application for geometric visualization


# Setup

##### Prepare js dependencies
`$ npm i`

##### Prepare cordova platforms and plugins
`$ cordova prepare`

##### To Start

<table>
<tr>
<td>Browser</td>
<td>

`$ npm run browser-dev`

</td>
</tr>
<tr>
<td>iOS</td>
<td>

`$ npm run ios-dev`

</td>
</tr>
</table>
