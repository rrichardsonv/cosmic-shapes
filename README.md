# cosmic-shapes
Simple canvas cordova application for geometric visualization


# Setup

_prepare js dependencies_
`$ npm i`

_prepare cordova platforms and plugins_
`$ cordova prepare`

### To Start

<table>
<tr>
<td>Browser</td>
<td>

`$ npm run browser-dev`

</td>
</tr>
<tr>
<td>iOS</td>
<td>

`$ npm run ios-dev`

</td>
</tr>
</table>
